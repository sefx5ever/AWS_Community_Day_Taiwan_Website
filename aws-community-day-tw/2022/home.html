    我們是誰？

    AWS User Group Taiwan (AWSUG-Taiwan) 是一個位於臺北，由 AWS Hero、AWS Builder、AWSUG 志工夥伴三大角色組成的 AWS 社群愛好者組織， 致力於 AWS 愛好者之雲端技術交流與分享，在不同產業的技術應用中達到最佳實踐的技術。

    AWS Community Day Taiwan 是一場從活動規劃、籌備到實踐，全由社群領袖發起的交流活動。活動將邀 請來自 AWS 的全球用戶的專家與產業領導，透過技術交流、研討會及工作坊，彼此相互交流與學習，同時 讓開發人員了解與認識 AWS 的最新技術與最佳實踐。

    歡迎所有產業主管、工程師、產品經理等參與每月由 AWSUG Taiwan 的交流會，或加入成爲志工或講師， 讓 AWS 技術能夠賦能您所屬的產業情景。
